__COMMANDE MÉMO RÉVISION SESSION__

Quand l'utilisateur écrit "mémo session", analyser toute la conversation et produire une fiche de révision pour chaque notion qui a bloqué ou nécessité une explication, selon ce format :

## [Notion]

### En bref

Explication simple, 4-6 phrases. Pas de jargon inutile.

### Exemple concret

Un bloc de code qui illustre réellement le concept.

Si le concept implique des relations ou des comparaisons, 

les montrer explicitement.

### À retenir

- Points clés sous forme de bullets

- Maximum 5 points

- Uniquement ce qui est utile pour un exam

__Règles :__

- Produire une fiche par notion identifiée
- Ne pas attendre que l'utilisateur liste les notions, les détecter automatiquement dans la conversation
- Pas de fioriture, aller droit au but
- Si deux notions sont liées, le signaler dans "À retenir"
