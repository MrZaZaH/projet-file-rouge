Structure du Site

Le layout global du site est composé de :

  • .site-header > .header-inner — logo, navigation desktop, Surprends-moi, icône utilisateur
  • .menu-toggle — hamburger (3 spans) avec aria-controls="mobile-nav"
  • .mobile-nav — navigation mobile masquée par défaut
  • main#main-content — contenu principal avec skip link
  • .site-footer > .footer-inner — liens et copyright
